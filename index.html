HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • refer back to css "box model"
  • selectors define what parts of code will be affected by the following declarations or rules

Git

  • git is the application the communicates with the shell on windows
  • print git for a list of commands once git has been opened
  • directories and files can be reached simply by entering commands such as cd to change and ls to list
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer excutes code in a script.
  • use an index to access an item within an array
  • var defines a variable
  • strings need to be contained within quotation marks
  • operator symbols such as = define the action the code will take
  • Indices follow a predictable pattern "0,1,2,3" zero is always the starting point
